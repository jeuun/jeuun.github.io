# scheduled (cron 시간 설정)
- 월요일 새벽 2시에 week 정보 등록
	 *  초    |   분   |   시   |   일   |   월   |  요일
	 *  0       0     2      *      *     MON
   - @Scheduled(cron = "0 0 2 * * MON")
   - log.debug(" !!! WEEK CREATED !!!");로 돌려서 실행이 잘 되는지 확인
